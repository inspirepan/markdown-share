# Markdown Share

A client-side Markdown editor that stores everything in the URL.

## Features

- No backend, no login required
- Content is compressed and encoded into the URL hash for easy sharing
- Also saves to localStorage so content persists on refresh
- Automatically follows system light/dark theme

## Usage

Visit https://inspirepan.github.io/markdown-share/

Edit your content, then copy the URL to share.
